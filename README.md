## ⚙️ Cài đặt

### Yêu cầu hệ thống
- Node.js version 18.x trở lên
- npm hoặc yarn
- Các yêu cầu khác (nếu có)

### Các bước cài đặt

1. Clone repository
https://github.com/nguyenhuykhai/football.git

2. Install dependencies
npm install

3. Run the project
npm run dev
